### Installation instructions

##### Copy the repository for yourself:

```sh
git clone <link_to_repository>
```

##### Install dependencies:

```sh
npm install
```

##### Create an .env file and copy the data from the .env.dev file.

##### Run the project:

```sh
npm run start
```
